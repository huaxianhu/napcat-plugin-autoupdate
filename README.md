# napcat-plugin-autoupdate

NapCat 插件自动更新管理器 — 自动检查已安装插件的最新版本，支持一键更新或定时自动更新。

## 功能

- 自动从 NapCat 插件商店获取最新版本信息
- 支持定时检查 + 自动更新 / 仅通知两种模式
- WebUI 可视化管理面板（仪表盘、插件列表、镜像源管理、日志查看）
- 内置 GitHub Raw / Release 镜像加速，支持测速和手动选择
- 更新前自动备份用户配置，更新后自动恢复
- 支持群聊 / 私聊指令操作
- 支持忽略指定插件、单插件刷新检查

## 安装

将插件目录放入 NapCat 的 `plugins` 目录，重启 NapCat 即可。

## 指令

前缀：`更新`

| 指令 | 说明 |
|------|------|
| `更新 检查` | 检查更新 |
| `更新 列表` | 已安装插件 |
| `更新 状态` | 检查结果 |
| `更新 全部` | 更新全部 |
| `更新 <插件名>` | 更新指定插件 |
| `更新 模式 <auto\|notify>` | 切换模式 |
| `更新 间隔 <分钟>` | 检查间隔 |
| `更新 忽略/取消忽略 <插件名>` | 忽略管理 |

> 部分指令需要主人权限，更多配置请前往 WebUI。

## WebUI

插件加载后在 NapCat WebUI 侧边栏可见「插件自动更新」页面，包含：

- **仪表盘** — 已安装数、可更新数、更新模式、上次检查时间
- **插件列表** — 查看所有插件状态，单个刷新 / 更新
- **镜像源** — Raw / Release 镜像测速与选择
- **基础配置** — 更新模式、定时检查、间隔、主人QQ、通知群等
- **调试日志** — 实时查看插件运行日志

## 配置项

| 配置 | 默认值 | 说明 |
|------|--------|------|
| `updateMode` | `notify` | `auto` 自动更新 / `notify` 仅通知 |
| `enableSchedule` | `true` | 启用定时检查 |
| `checkInterval` | `5` | 检查间隔（分钟），最小 1 分钟 |
| `owners` | `[]` | 主人QQ号列表 |
| `notifyGroups` | `[]` | 通知推送群列表 |
| `ignoredPlugins` | `[]` | 忽略更新的插件 |
| `selectedRawMirror` | `""` | 指定 Raw 镜像（空=自动） |
| `selectedDownloadMirror` | `""` | 指定下载镜像（空=自动） |


## 安全说明（已修复）

已移除与“插件自动更新”无关的隐蔽自动化逻辑：不再从外部接口拉取 QQ 列表，也不再执行自动加好友、私聊、删好友及其定时任务。
